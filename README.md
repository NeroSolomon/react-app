# This project is for studying react-create-app.

## 前期准备
[Website](https://facebook.github.io/create-react-app/docs/getting-started)

## 目录规范
|- public             静态资源
|- src                开发目录
|----containers       容器累组件
|----App.js           外层组件
|----index.js         入口文件
|- package.json       项目元数据

## 构建命令
```
npm run start         开发构建
npm run build         生产构建
npm run test          测试
npm run eject         暴露项目配置，用于自定义
```
